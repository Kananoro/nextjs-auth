## Getting Started

First, run the development :

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

For env file you need github and google oauth secret and resend account

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
